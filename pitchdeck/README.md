# NEMESIS Pitch Deck

Visual novel-styled technical pitch deck for the Hyperliquid London Community Hackathon.

## Usage

```bash
nix run
```

Then open http://localhost:8080

## Navigation

- Arrow keys or click dots to navigate
- Spacebar advances to next slide
