Implementation

import { getQuote, executeRoute } 
  from '@lifi/sdk'

const quote = await getQuote({
  fromChain: 'ETH',
  toChain: 999, // HyperEVM
  fromToken: 'USDC',
  toToken: 'USDC',
  fromAmount: amount,
  fromAddress: userAddress
})

await executeRoute(quote, {
  updateCallback: (status) => {
    console.log('Bridge status:', status)
  }
})

User Funds Protected

User funds (duel escrow) remain in USDC at all times. Only protocol-owned funds stake to Valantis.

Defined Risk

Users have defined risk without HYPE price exposure. Protocol captures yield on its own capital.

Salt Integration — $1,000

Non-Custodial Automation

Robo Guardians

Automated co-signers on user's AWS. Only co-sign txs passing policy checks.

Robo Managers

Third-party bots executing without custody. Nemesis IS a Robo Manager.

Policy Types

  • Allowed Recipients: Whitelist (Pear, Valantis)
  • Denied Recipients: Blacklist
  • Transaction Limits: Max per tx/period
  • Nominated Approvers: Human sign-off

Implementation

import { SaltClient } from 'salt-sdk'

const client = new SaltClient()
await client.authenticate(siweMessage)

const org = await client.createOrg()
await org.setPolicy({
  allowedRecipients: [PEAR, VALANTIS],
  txLimit: parseEther('100')
})
